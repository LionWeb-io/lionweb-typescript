This NPM package holds a code generator to generate classes based on the `class-core` package from an M2.


## Installation

Run the following command to add this package to an NPM-based project:

```shell
npm add @lionweb/class-core-generator
```

This adds this package as a dependency to your NPM-based project.


## Usage

* Use the `generateLanguage`, `generateApiFromLanguages`, `generateApiFromLanguagesJson`, and `megaFactoryFor` functions to generate API code for a single language or any number of languages.
* The `defaultTrumpfOriginatingApache2_0LicensedHeader` and `generatedCodeWarningComments` text constants can be used as file headers.

As always: see the JSDoc of these items for more details.
