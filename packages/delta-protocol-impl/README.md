# The `delta-protocol-impl` package

[![license](https://img.shields.io/badge/License-Apache%202.0-green.svg?style=flat)
](./LICENSE)

An NPM package that can be added to a Node.js/NPM codebase as follows:

```shell
$ npm add @lionweb/delta-protocol-impl
```
It contains the (production part of the) implementation of the delta protocol in TypeScript, using the WebSocket protocol.


## Development

Build it from source as follows:

```shell
npm run build
```
