# The `delta-protocol-test` package

[![license](https://img.shields.io/badge/License-Apache%202.0-green.svg?style=flat)
](./LICENSE)

An (unpublished) NPM package that can be added to a Node.js/NPM codebase as follows:

```shell
$ npm add @lionweb/delta-protocol-test
```
It contains all tests pertaining to the `delta-protocol-impl` package.


## Development

Build it from source as follows:

```shell
npm run build
```
