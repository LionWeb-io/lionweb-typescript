# Definitions

The files in this folder contain the definitions for the format of the delta and bulkapi messages.

The definitions are in the format used by the Validator in this package and used to validate the structure of delta messages and bulkapi requests.

**NB.** These files are generated from a LionWeb model (will add reference later on) and should not be changed manually. 

## ChunkDefinitions
Contains all the type definitions for the LionWeb serialization format as described in the specification.

## Command, Event, Query, Request, Response - definitions
These files contain the type definitions for all the delta messages in the LionWeb specification.
For easy cross-reference, the TSDoc of each delta contains a link to the place in the LionWeb specification where the delta is defined.

## DeltaType Definitions
This file contains the type definitions for the types of the properties being used in the Delta ,messages.
